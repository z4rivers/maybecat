# Plan 05-01: Extract decorative SVG components

## Goal

Move decorative SVG components from Home.tsx to dedicated files in `src/components/decorative/`.

## Why

- These are pure presentational components with no logic
- Moving them reduces Home.tsx size significantly (~70 lines)
- Makes them reusable if other pages need the same decorations

## Components to Extract

### CornerVine (lines 11-44)

- Props: `{ className?: string }`

### CenterMandala (lines 46-69)
- Ornate mandala-style center decoration
- ~24 lines
- Props: `{ className?: string }`

### MysticalStar (lines 72-79)
- Simple 8-pointed star
- ~8 lines
- Props: `{ className?: string }`
