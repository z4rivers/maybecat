# Plan 03-01: Create config file and replace magic numbers

## Goal

Extract magic numbers from Home.tsx to a centralized config file.

## Why

Magic numbers scattered throughout code are harder to find and update. A config file:
- Makes tuning behavior easier (change one place)
- Documents what each value means
- Prevents accidental inconsistencies

## Config Structure

```typescript
// src/config.ts
export const config = {
  // Image brightness analysis
  brightness: {
    sampleSize: 100,
    threshold: 70,
    enhanceFilter: 'brightness(1.3) contrast(1.1)',
  },

  // Shelter cats
  shelterCats: {
    fetchCount: 5,
  },

  // Oracle "thinking" delays (milliseconds)
  thinking: {
    firstAsk: { base: 1500, variance: 1500 },    // 1.5-3s
    askAgain: { base: 800, variance: 800 },       // 0.8-1.6s
  },

  // Image export
  export: {
    scale: 2,
  },
};
```
